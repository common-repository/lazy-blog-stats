== Description ==

A simple plugin that shows a count of how many posts, comments and users your blog has.

Really that is all it does with a simple configuration screen. 


== Installation ==

1. Upload the `LazyBlogStats` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php if(function_exists('ShowBlogStats')) { ShowBlogStats();}?>` in your templates
